# Curl and Divergence ## Curl Curl is the measure of how much the field rotates, meaning that a particle will rotate as it travels through the field. Clockwise is negative adn counter-clockwise is positive. If curl is 0, the particle does not rotate, but may move. Imporantly, if curl is 0, then the vector field is conservative, and it can be integrated, meaning that there is an $f$ such that $\nabla f = \vec F$. $$Curl = \left({\delta R \over \delta y} - {\delta Q \over \delta z}\right)\hat i + \left({\delta P \over \delta z} - {\delta R \over \delta x}\right)\hat j + \left({\delta Q \over \delta x} - {\delta P \over \delta y}\right)\hat k$$ This is a bit of a pain to remember, so let's use a more convenient formula. Recall that $$\nabla f = \langle {\delta f\over \delta x}, {\delta f\over \delta y}, {\delta f\over \delta z}\rangle$$ We can show $\nabla$ as a special vector of derivatives and this as a scalar product. $$\nabla = {\delta \over \delta x}\hat i + {\delta \over \delta y} \hat j + {\delta \over \delta z}\hat k$$ Using this notation, curl can be defined as: $$curl\ \vec F = \nabla \times \vec F$$ By computing the cross product of this we get back to the same formula, but through a much easier-to-remeber way. ## Divergence Divergence is the measure of how much the field pushes the particles away from their starting point. Ex: an explosion would be higly positive, a black hole would be higly negative. If the particle does not move, divergence is = 0. Similarly to curl, divergence can be defined using $\nabla$: $$div\ \vec F = \nabla \cdot \vec F$$ This is a bit simpler to compute: $$div\ \vec F = {\delta P\over\delta x} + {\delta P\over\delta y} + {\delta P\over\delta z}$$ ### Theorem If $\vec F$ is a vector field and P,Q,R have continuous second pertial derivatives, then: $$div\ curl\ \vec F = 0$$
